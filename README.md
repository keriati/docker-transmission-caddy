# Scrutiny with Caddy

Use Scrutiny with green https on your local network.

Requirements
- Own domain
- Cloudflare DNS setup for your domain
- Cloudflare API Token with zone and dns access

Environment variables:

- WHITELIST
- ADMIN_USER
- ADMIN_PASS
- TRANSMISSION_DOMAIN
- APP_DIR
- FILE_DIR
- CLOUDFLARE_EMAIL
- CLOUDFLARE_API_TOKEN
